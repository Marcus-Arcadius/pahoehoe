Come funziona Lavabit Encrypted Proxy?

Con il Lavabit Encrypted Proxy utilizziamo la tecnologia Virtual Private Networking (VPN) per fornire una pipeline crittografata per tutto il tuo traffico Internet. È un tunnel sicuro tra il tuo computer e la dorsale Internet. Oltre ogni ficcanaso che potrebbe non essere niente di buono. Dai nostri server puoi accedere a Internet senza censura o sorveglianza. Inoltre, i siti Web e i server a cui accedi non vedranno più la tua identità, o meglio il tuo indirizzo IP, ma vedranno invece l'indirizzo proxy generico che forniamo.
